# CodeAnalyzer

Aplicación web que analiza código con inteligencia artificial. Detecta errores potenciales, sugiere mejoras y explica el funcionamiento del código en lenguaje natural.

## Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Python, FastAPI
- **IA:** LangChain, Ollama (modelo local)

## Requisitos previos

- Python 3.10+
- [Ollama](https://ollama.com) instalado y en ejecución
